## Ciclo de Vida del Componente

\`\`\`
1. MONTAJE
   в”ңв”Җ Componente recibe props (section, animated, children)
   в””в”Җ useEffect se ejecuta

2. GENERACIГ“N
   в”ңв”Җ Llama generateSectionBackground(section)
   в”ңв”Җ Lee SECTION_PATTERNS[section]
   в”ңв”Җ Genera SVG dinГЎmico
   в””в”Җ Retorna { svg, color, opacity }

3. ESTADO
   в”ңв”Җ setSvgContent(svg)
   в”ңв”Җ setColor(color)
   в””в”Җ setOpacity(opacity)

4. RENDER
   в”ңв”Җ Renderiza <div> contenedor
   в”ңв”Җ SVG en capa de fondo (pointer-events-none)
   в”ңв”Җ Overlay gradiente adicional
   в”ңв”Җ Contenido en capa de contenido (z-10)
   в””в”Җ Aplicar animaciones CSS si animated={true}

5. ANIMACIГ“N
   в”ңв”Җ CSS @keyframes float se ejecuta
   в”ңв”Җ DuraciГіn: 8s
   в”ңв”Җ RepeticiГіn: infinite
   в””в”Җ Loop continuo suave
