## Common Questions Answered

### Q: How long until Google indexes everything?
**A:** 2-4 weeks for most pages. Some may take longer depending on content depth.

### Q: Will both Spanish and English rank equally?
**A:** Spanish ranks better initially (default). English versions will follow with proper hreflang.

### Q: What if a page isn't indexed?
**A:** Use URL Inspection tool in Search Console. If blocked, check robots.txt or meta robots tag.

### Q: How often does the sitemap update?
**A:** Automatically with each deployment. No manual updates needed.

### Q: Will old pages be re-indexed?
**A:** Yes, once you submit the sitemap. Google will re-crawl everything.

### Q: What about images and PDFs?
**A:** Images are indexed if they have alt text and proper URLs. This setup covers HTML pages.

## Emergency Troubleshooting

```
If indexing stalls:
├─ Check robots.txt isn't blocking URLs
├─ Verify no "noindex" tags
├─ Ensure pages return 200 status
├─ Check Core Web Vitals
└─ Inspect page in Search Console

If getting 404s:
├─ Check URL structure in sitemap
├─ Verify redirects are 301 (permanent)
├─ Test URLs manually in browser
└─ Check server logs
```
