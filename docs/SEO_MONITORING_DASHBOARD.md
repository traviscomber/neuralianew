### Key Metrics to Track

- **Total Impressions**: How many times your site appears in search
- **Total Clicks**: How many people click from search results
- **Average CTR**: Clicks ÷ Impressions (aim for >2-3%)
- **Average Position**: Lower is better (aim for <15 for high-value keywords)

### Red Flags (Take Action If)
- CTR drops >20%: Check if snippets were removed (schema issue?)
- Clicks drop >30%: Check if pages were deindexed or ranked dropped
- Impressions drop >20%: May indicate content quality issue or algorithm change
- Position drops >5: Competitor ranking up, need content optimization

## ACTION THRESHOLDS

### IF: Organic Traffic Drops >20%
**Possible Causes**:
- Algorithm change (check Google Search Central blog)
- Deindexing (check GSC coverage report)
- Keyword ranking drops (check top 25 keywords)
- Technical issue (check Core Web Vitals, 404 errors)

**Action**:
1. Check GSC for coverage issues
2. Review PageSpeed Insights
3. Compare rankings to last month
4. Review recent changes to site
5. If algorithm-related: Wait + improve content quality

### IF: CTR Drops But Position Stays Same
**Possible Causes**:
- SERP snippets changed (missing yours)
- Rich results disappeared
- Ad taking up space (less room for organic)
- Title/meta description not compelling

**Action**:
1. Check SERP appearance in GSC (before/after)
2. Improve title/meta description
3. Check schema.org markup
4. A/B test new meta descriptions

### IF: Keyword Enters Top 20 But Plateaus
**Possible Causes**:
- Page content good but not best
- Competitor content better
- Internal linking weak
- Backlink profile weak

**Action**:
1. Analyze top 3 competitors' content
2. Improve your page (longer, more detailed, better structure)
3. Add internal links from authoritative pages
4. Seek high-quality backlinks

### IF: New Traffic Source Appears (Unlabeled)
**Possible Causes**:
- New keywords ranking
- Viral social share
- Press mention
- Content appearing in AI (Perplexity, Claude)

**Action**:
1. Investigate in GSC (find source keywords)
2. Optimize landing page for this traffic type
3. If AI mention: Verify schema.org markup is correct
4. Consider creating more content for this angle
