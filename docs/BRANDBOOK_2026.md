## 1. Brand Positioning

N3uralia builds production AI systems, agentic workflows, and software automation for teams in Chile and LATAM.

The brand should feel:

- Precise, not decorative
- Technical, not generic SaaS
- Cinematic, not flashy
- Industrial, not playful
- Human-controlled, not autonomous by default
- Premium, but never ornamental

The visual identity is best described as **retro-futurist operational intelligence**.

---

## 2. Core Visual Principles

### 2.1 Dark operational environment

The primary brand surface is near-black. It represents control rooms, active systems, telemetry, and live operations.

### 2.2 Teal signal language

Cyan-teal is used as a system signal: active states, focus, controls, status, traces, lines, scanning effects, and technical emphasis.

### 2.3 Editorial contrast

Pale cyan light sections are used intentionally to create rhythm, explain outcomes, and separate narrative layers. They are not the default application background.

### 2.4 Structured asymmetry

Layouts should use deliberate imbalance: wide image fields, narrow copy columns, offset grids, split panels, sticky introductions, and horizontal systems diagrams.

### 2.5 Restrained motion

Motion should communicate state, direction, scanning, or system activity. It must not exist only for decoration.
